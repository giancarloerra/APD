✨ Features

🌙 Weather Dashboard
  • Dual-source: Meteoblue + Met Office
  • Night-only hourly forecasts
  • Astrophotography score (0–100)
  • 7-day expandable day rows
  • Hourly cards: cloud, temp, rain, wind
  • Cloud cover sparkline per night
  • Weekly cloud cover heatmap
  • Best night banner
  • Moonlight % & sky brightness
  • Solar astrophotography section
  • Live SDO solar imagery
  • Auto/manual weather refresh
  • LLM-friendly summary endpoint (public API)
🔭 Sky Dashboard
  • Interactive sky map (Aladin Lite)
  • Framing mode + Sky Chart mode
  • Camera FOV overlay & lock
  • Mosaic planner (N×M tiles)
  • Camera rotation / angle offset
  • Saved FOVs & favorite targets
  • Target search by name
  • Time travel (±h / d / month)
  • Constellation, star & DSO overlays
  • Moon, planets & red night mode
  • Horizon line & grid overlays
  • Discover panel (Telescopius API)
